# Casino Simulation CLI
CLI-приложение на Python для моделирования работы казино: запуск симуляции с разными параметрами.

## Описание проекта
Проект реализует объектную модель казино с игроками, гусями, ставками и другими разными событиями и предоставляет интерфейс командной строки для управления симуляцией.​

### Основные возможности
Запуск симуляции казино с настраиваемыми параметрами (количество игроков, гусей).​
Описание сущностей казино (игроки, гуси) через отдельные классы и коллекции.​
Сбор статистики по результатам симуляции (выигрыши/проигрыши игроков, атаки гусей).​
Управление логикой сценариев через отдельный модуль симуляции.​​

## Установка
```bash
git clone https://github.com/SOKLY6/Lab_4.git
cd Lab_4
```

### Установка зависимостей
```bash
pip install uv
source .venv/bin/activate
uv sync
```

## Запуск
```bash
python3 main.py
```
